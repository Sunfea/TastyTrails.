# TastyTrails

A comprehensive food delivery platform built with Django and FastAPI.

## Features
- Restaurant management and discovery
- Menu browsing and ordering
- User authentication and profiles
- Delivery tracking
- Reviews and ratings
- Search functionality

## Tech Stack
- **Backend**: Django 5.2, FastAPI
- **Database**: PostgreSQL
- **Caching**: Redis
- **Background Tasks**: Celery
- **Frontend**: Django Templates with Bootstrap
- **Deployment**: Docker, Railway-ready

## Deployment
This application is configured for deployment on Railway. For detailed deployment instructions, please refer to the Railway deployment configuration files.

## Key Components
- User authentication and profile management
- Restaurant and menu management
- Order processing and delivery tracking
- Review system for restaurants and dishes
- Search functionality for restaurants and cuisines

## Development
To run locally:
1. Install dependencies from requirements.txt
2. Configure environment variables
3. Run Django migrations
4. Start the development server
